# Code Of Conduct

## Our Standard

Contributors are expected to keep discussions respectful, factual, and focused
on improving the project. Disagreement is acceptable; harassment, threats,
personal attacks, and discriminatory behavior are not.

## Scope

This code of conduct applies to project spaces including issues, pull requests,
discussions, documentation, and other collaboration channels connected to this
repository.

## Enforcement

Project maintainers may remove comments, close discussions, or restrict
participation when behavior violates this standard.

If you need to report a conduct issue, contact the repository owner with a clear
description and links to relevant public context where possible.
